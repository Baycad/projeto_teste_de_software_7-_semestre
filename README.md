# 🏢 Sistema de RH – Projeto de Teste de Software

Sistema web de gestão de Recursos Humanos desenvolvido como projeto prático da disciplina de **Teste de Software – 7º Semestre**.

O sistema será utilizado em sala para aplicação de técnicas de teste, validação de funcionalidades e identificação de possíveis falhas.

---

## 🎯 Objetivo do Projeto

O objetivo principal deste projeto é:

- Desenvolver um sistema web funcional para gerenciamento de funcionários
- Aplicar conceitos de Teste de Software na prática
- Executar testes funcionais, exploratórios e de validação em sala
- Identificar defeitos e propor melhorias

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- LocalStorage (persistência de dados no navegador)  
- Git e GitHub  
- Netlify (deploy)

---

## ⚙ Funcionalidades Implementadas

✔ Cadastro de funcionários  
✔ Listagem dinâmica  
✔ Busca por nome ou CPF  
✔ Filtro por departamento  
✔ Exclusão de funcionário  
✔ Dashboard com métricas  
✔ Persistência de dados no navegador  

---

## 🧪 Contexto de Testes

O sistema será utilizado para aplicação prática de:

- Testes funcionais  
- Testes de validação de formulário  
- Testes de entrada inválida  
- Testes de usabilidade  
- Testes exploratórios  
- Identificação e documentação de bugs  

Durante a aula, serão analisados:

- Comportamento do sistema  
- Tratamento de erros  
- Validações  
- Consistência dos dados
